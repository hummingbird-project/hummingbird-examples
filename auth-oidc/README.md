# Open ID

Example demonstraing login with OpenID.

Example requires the following environment variables to be set
- OIDC_CLIENT_ID: client id for OIDC application
- OIDC_CLIENT_SECRET: client secret for OIDC application
- OIDC_DISCOVERY_URL: URL to discover OIDC configuration
- OIDC_REDIRECT_URI: Application endpoint to redirect to during authorization
